# Premium Font Palettes

Use these curated pairings to provide high-quality, "Antigravity-level" design suggestions.

## 1. The Modern Professional (SaaS/Tech)
- **Heading**: **Inter** (SemiBold, 600)
- **Body**: **Inter** (Regular, 400)
- **Vibe**: Clean, objective, highly readable. The gold standard for modern web apps.
- **CSS**:
```css
:root {
  --font-heading: 'Inter', sans-serif;
  --font-body: 'Inter', sans-serif;
}
```

## 2. The Elegant Storyteller (Creative/Portfolio)
- **Heading**: **Playfair Display** (Black, 900)
- **Body**: **Source Sans Pro** (Regular, 400)
- **Vibe**: Sophisticated, editorial, premium. Great for high-end brands.
- **CSS**:
```css
:root {
  --font-heading: 'Playfair Display', serif;
  --font-body: 'Source Sans Pro', sans-serif;
}
```
